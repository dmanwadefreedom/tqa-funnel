Design Decisions

Hierarchy (2-second rule): 1. What he does (headline) → 2. Why trust him (credentials) → 3. What to do (CTA). DJ's photo is present but subdued so it validates without taking over.

Messaging: "Are You Cybersecurity-Ready?" speaks to both audiences — IT pros wondering if they can make the switch AND security managers wondering if they're ready for the C-suite. One question, two buyers.

CTA: "Free Readiness Score" not "Free Career Quiz" — specific, outcome-oriented, speaks to the assessment they'll actually get.

Style: Trust & Authority (UI/UX Pro Max recommendation). Navy + gold. Poppins headings + Inter body. No cyberpunk, no neon, no gaming aesthetic. A CISO presenting, not a hacker coding.

Photos: Speaking on stage and award ceremony — both show authority in action, not a static headshot. Faded so they don't compete with text.

Logos: Small, muted, bottom of hierarchy. They're proof signals, not the headline.

To export: Screenshot each banner at 2x resolution, or rebuild in Canva/Figma using these as the spec. The photos reference local file paths — swap for uploaded versions in the design tool.
